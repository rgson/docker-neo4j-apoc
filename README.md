# Neo4j with APOC

Docker container with [Neo4j] and [APOC].

The image is based on Neo4j's [official Docker image][dockerhub-neo4j], but
comes with the APOC plugin [installed and configured][github-apoc].

[Neo4j]: https://neo4j.com/
